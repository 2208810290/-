# 京东试用

京东试用脚本完善版，增加了青龙支持、多用户支持、ck检测、日志记录、优化多请求等

目前自动申请手机、电脑相关商品

青龙环境变量名称：jd_ck

格式：pt_key@pt_pin

例如:pt_key=123,pt_pin=456

格式就是：123@456

多用户请用#分割，例如：123@456#789@1011
